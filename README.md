### Description

Welcome to my rendition of The Cloud Resume Challenge. Here you will find all the necessary files to configure and set up a static site on S3 using terraform. 

### Instructions

1. Install Terraform
2. Update variables etc. to match your needs
3. terraform init
4. terraform plan
5. terraform apply
